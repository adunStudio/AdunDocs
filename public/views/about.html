Open-document notes that support Tistory & MarkDown

AdunDocs is Open-document notes that support Tistory & Markdown (of course HTML) and don't need DB

  • Created and maintained by adunStudio
  • Free and open source

To keep up-to-date with the latest news:

  • Visit OppaCoding on Website
  • Visit 소년코딩 on Tistory

If you like AdunDocs, please consider giving a on my Git.   Thanks!

Features

1. Don't need DB

  • All files are stored on local disk
  • You don't need complex DB setting.

2. Support MarkDown

  • Markdown is a lightweight markup language that can be converted to HTML
  • You will easily edit the documents.
  • Of course, also provides HTML editor.

3. Support Blog

  • AdunDocs support blog editing
  • You will edit blog posts with MarkDown
  • Available blog
    Blog Features
    Tistory getUsersBlogs, getRecentPosts, getCategories, getPost, editPost, newPost, deletePost, newMediaObject
    Naver getRecentPosts, getCategories, getPost, editPost, newPost, newMediaObject
